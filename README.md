# Women's Basketball Coaching Hotboard

A comprehensive coaching intelligence platform for NCAA Women's Basketball.

## Features

- **By School**: Select a school to see current staff and their coaching connections
- **By Alma Mater**: Find coaches by where they played college basketball  
- **By Coach**: Search for any coach by name
- **By Stats**: View coaches ranked by T-Rank performance metrics
- **Staff History**: Year-by-year staff breakdowns for any program
- **School History**: Historical coaching staff for programs

## Coach Profiles

Three-tab modal system:
- **Career**: Full coaching history with HC positions highlighted
- **Tree**: Coaching tree connections (worked under / mentored)
- **Stats**: T-Rank performance data (2021-2026)

## Data Sources

- Coach data: Wikipedia + school athletic websites
- T-Rank data: barttorvik.com (2021-2026)

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- React + Vite
- Tailwind CSS (utility classes)
- Team logos via ESPN CDN
